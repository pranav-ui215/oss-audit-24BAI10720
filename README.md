# oss-audit-24BAI10720
Audit of LibreOffice with 5 Linux shell scripts and open-source analysis.

## Overview
This project explores LibreOffice as an open-source software system. It includes analysis of its origin, license, ecosystem, and comparison with proprietary software.

## Scripts Included
1. System Identity Report
2. FOSS Package Inspector
3. Disk and Permission Auditor
4. Log File Analyzer
5. Open Source Manifesto Generator

## Technologies Used
- Linux (WSL Ubuntu)
- Shell Scripting (Bash)
